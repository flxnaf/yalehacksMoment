Processing pipeline

Every frame through
four stages.

Camera frames flow through a fully asynchronous pipeline — each stage is rate-limited and isolated so no component can stall the others.

01

Camera Capture

Live frames at 30fps from either the iPhone AVCaptureSession or Meta Ray-Ban glasses via MWDAT SDK and WebRTC. Frames are timestamped and routed to both the depth engine and Gemini Live simultaneously.

02

Depth + Vision

DepthAnything V2 (CoreML, ANE-accelerated) produces a 448×448 monocular depth map per frame. Apple Vision runs person detection at 5Hz and scene classification at 1Hz in parallel, cross-referenced with depth values per bounding box.

03

Obstacle Analysis

ObstacleAnalyzer samples depth in left/center/right zones using robust 5th-percentile values. PathFinder scans a 24-column horizontal depth profile for walkable gaps. NavigationAudioPolicy fuses urgency with hysteresis filtering.

04

Audio + Guidance

SpatialAudioEngine places obstacle pings and the navigation beacon in 3D HRTF space anchored to ARKit world coordinates. Priority-queued speech (hazard > navigation > ambient) delivers ElevenLabs TTS or Gemini Native Audio to the user.

Features

Twelve systems,
one continuous experience.

Every component runs concurrently — on device, with no internet required for the core navigation loop.

Core navigation

Shrine Ping Beacon

Ask Gemini to guide you anywhere — "take me to the counter." A spatial audio beacon locks to a real GPS coordinate and ARKit world anchor. A Zelda-style chime pair (C6 + E6) plays from that exact 3D direction via HRTF. Gyro + compass fusion keeps "12 o'clock" pointing straight ahead regardless of rotation. Auto-clears on arrival within 3m.

HRTF · AVAudioEnvironmentNode GPS + ARKit anchor Gyro-relative
Core navigation

Turn-by-Turn Navigation

Say "take me to the library" — Gemini calls the navigate_to tool, Google Maps Directions API computes a pedestrian route, and FusedNavigator guides you waypoint-by-waypoint at 10Hz. Obstacle urgency from depth analysis feeds directly into bearing decisions — it will tell you to detour before you reach something.

Google Maps Directions API GPS + compass EMA 10Hz fusion loop

Gemini 2.5 Live Voice

Persistent WebSocket to Gemini 2.5 Flash Native Audio Preview. Bidirectional PCM audio (16kHz mic, 24kHz output) + JPEG video every second. Natural voice conversation with full tool-call routing — set beacons, start navigation, ask what's ahead.

WebSocket · 16kHz/24kHz Tool routing

Proactive Obstacle Scans

When a beacon is active or depth urgency exceeds 0.25, Gemini automatically scans the live camera and announces what's ahead: "Chair at 12 o'clock. Wall to the right." 5-second cooldown with beacon, 8-second otherwise. Never fires mid-sentence.

Urgency-gated Adaptive cooldown

Parking Sensor Audio

Silent when the path is clear. Escalating DSP-synthesized beeps as objects enter a 2m danger zone. Left, center, and right zones each have independent tone frequency, rate, and spatial pan — no noise pollution when walking freely.

AVAudioEngine DSP Zone-independent

Monocular Depth

DepthAnything V2 Small runs fully on Apple's Neural Engine — no cloud, no privacy concerns. Two-phase loading: GPU (5s cold start) then hot-swap to ANE once compiled. Turbo colormap LUT via Accelerate's vDSP_vfixu8. ~43ms per frame.

CoreML · ANE Hot-swap backends

Person Detection

VNDetectHumanRectanglesRequest at 5Hz, cross-referenced with depth (80th-percentile per bbox). SmartAssistant announces clock-face direction. Major hazard threshold (depth > 0.956, ~0.8m) bypasses cooldown and interrupts all other speech immediately.

Vision.framework Depth-fused

Walkable Path Detection

PathFinder scans a 24-column depth profile across the eye-level image band. Floor pixels are excluded via vertical gradient analysis. ClearPath objects score by width × (1 − avgDepth) and map directly to haptic and spatial audio cues.

24-column profile Floor-filtered · EMA

Fall Detection + SOS

Three-phase CoreMotion state machine: freefall (<0.55g for ≥0.28s) → impact (>2.2g within 0.85s) → stationary confirmation. 10-second countdown with voice prompts. Guardian alerted via email (SendGrid), SMS, and WhatsApp (OpenClaw) with GPS + photo.

CoreMotion 100Hz SendGrid · WhatsApp

Priority Speech Engine

AudioOrchestrator manages a four-level queue: ambient → social → navigation → hazard. ElevenLabs TTS (voice "George", eleven_turbo_v2_5) for hazard priority with 12 pre-warmed phrases cached in memory at startup. Falls back to AVSpeechSynthesizer. Auto-resamples format mismatches.

ElevenLabs TTS Pre-warmed cache

Haptic Navigation

NavigationHapticEngine encodes walkable path direction and clearance distance into CoreHaptics patterns. Urgency-scaled feedback — light taps for clear paths, sharp pulses when an obstacle is immediately ahead. Operates independently of audio for deaf-blind users.

CoreHaptics Urgency-scaled

On-device depth

Seeing distance
without cloud.

DepthAnything V2 Small runs entirely on Apple's Neural Engine. Each frame is resized to 448×448, passed through the model, and produces a normalized depth map — 0 = far, 1 = near — in under 50ms on any modern iPhone.

The raw float map is colorized with a 256-entry Turbo LUT using Accelerate's vDSP_vfixu8 for a debug overlay. The same raw values drive every audio decision. The model hot-swaps between GPU and ANE backends without interrupting inference.

Gemini 2.5 Live

Always listening,
always watching.

A persistent WebSocket to gemini-2.5-flash-native-audio-preview carries bidirectional PCM audio (16kHz mic, 24kHz speaker) and JPEG video frames at 1fps. The model maintains full spatial context between turns.

Four tool calls give Gemini direct control over the app: navigate_to starts a GPS route, set_ping places the ARKit beacon, invoke_openclaw_tool routes to the OpenClaw gateway, and the [OBSTACLE SCAN] trigger fires proactively when depth urgency spikes above 0.25.

StreamSessionViewModel.swift 
// Proactive obstacle scan trigger
let shouldScan =
  (obstacle.urgency > 0.25 || audioEngine.beaconActive)
  && now.timeIntervalSince(lastScan) >= cooldown
  && !(geminiVM?.isModelSpeaking ?? false)

if shouldScan {
  lastObstacleScanTime = now
  geminiSessionVM?.sendObstacleScan()
}
Spatial audio

A 3D soundscape
around you.

SpatialAudioEngine places every sound in precise 3D space via AVAudioEnvironmentNode with HRTF-HQ rendering (iOS 15+). The 8-voice pool supports simultaneous obstacle pings, beacon chime, shore ambient, and Gemini voice — all spatially separated.

The shrine beacon is anchored to a real GPS coordinate and ARKit camera transform. EMA-smoothed compass heading ensures "12 o'clock" always means straight ahead. Shore ambient ocean sound layers in when you're within ±20° of the beacon direction.

SpatialAudioEngine.swift 
let env = AVAudioEnvironmentNode()
env.listenerPosition = AVAudio3DPoint(
  x: 0, y: 0, z: 0
)

// GPS-anchored bearing → 3D position
beacon.position = AVAudio3DPoint(
  x: sin(bearingRad) * dist,
  y: 0,
  z: -cos(bearingRad) * dist
)
